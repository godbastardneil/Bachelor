Запуск сервера:
    pm2 start server.js


Генерация и установка сценария запуска systemd PM2:
(при перезапуске компа, сервер сам бует запускаться)
    sudo pm2 startup


Просмотр списка процессов PM2:
    pm2 list


Просмотр информации о запущенных процессах:
(0 - ID процесса)
    pm2 show 0


Перезапуск процесса приложения:
(например, при внесении изменений)
(0 - ID процесса)
    pm2 restart 0
